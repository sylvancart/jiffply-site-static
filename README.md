# Jiffply Static Website

This repository contains the static website for Jiffply. It includes the main landing page, pricing details, feature overview, and contact page.

## Project Structure

- `index.html` - Home page for Jiffply.
- `features.html` - Description of product features and capabilities.
- `pricing.html` - Pricing plans and subscription details.
- `contact.html` - Contact information and inquiry form.
- `assets/css/style.css` - Site styles and layout rules.
- `assets/js/main.js` and `assets/js/main-new.js` - Website scripts and interactions.
- `assets/images/` - Image assets used across the site.

## Purpose

This repository is designed to serve a simple promotional website for Jiffply. The site is built using standard HTML, CSS, and JavaScript, and is intended for deployment as a static site.

## Local Testing

1. Open `index.html` in a browser.
2. Or serve the folder from a local web server to test navigation and script behavior.

## Notes

- Keep markdown files out of version control by using the `.gitignore` file.
- Use the existing CSS and JavaScript files to update styling or page behavior.
- Add or modify pages as needed to match site content.
